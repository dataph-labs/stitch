# stitch
=========

A package manager for most UNIX systems. Built for NeedleBSD.
https://github.com/dataph-labs/needlebsd

List of 100% supported systems:
- NeedleBSD
- NetBSD
- Linux

Build and Install.
------------------

NeedleBSD:
    
    Built into the base system

NetBSD:

    make
    su
    make install
    exit

Linux:

    make -f Makefile.linux
    sudo make install -f Makefile.linux
