# stitch
=========

A package manager for most UNIX systems. Built for NeedleSystem.
https://github.com/dataph-labs/needlesystem-src

List of 100% supported systems:
- NeedleSystem
- NetBSD
- Linux

Build and Install.
------------------

NeedleSystem:
    
    Built into the base system

NetBSD:

    make
    su
    make install
    exit

Linux:

    make -f Makefile.linux
    sudo make install -f Makefile.linux
